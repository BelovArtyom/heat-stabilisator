Программа, поддерживающая стабильную температуру, введённую пользователем, через управление поддувом при помощи вентилятора, добавлением топлива (угля) через определённый интервал времени.

Возможные дополнения:

    • Логи для отслеживания изменений в температуре и действий программы
    • Графический интерфейс для упрощения восприятия информации
    • Самообучаемость программы за счёт анализа изменений температуры по времени
    • Возможность оповещения указанного номера при нештатных ситуациях.
